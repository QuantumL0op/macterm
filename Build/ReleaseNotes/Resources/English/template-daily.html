Download Free Update (BETA) (version 4.1.0, under development; requires Mac OS X 10.9+)

Mac OS X 10.9 "Mavericks" is a free update from Apple (via the App Store) for users of Mac OS X 10.6, 10.7 and 10.8; therefore, MacTerm now requires 10.9+ instead of 10.6+.

Note:  You may also download 4.0.0, the last version to support Mac OS X 10.5 and earlier.
